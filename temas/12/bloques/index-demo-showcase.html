Features

Ruby Vertical Menu

Not every website is built on horizontal menus. Furthermore, in some pages vertical menus can be useful for better layouts. Ruby Mega Menu can be converted to a vertical menu with just one simple step:

Add .ruby-vertical class to wrapper and menu. So simple.

Features

Mobile Responsive. Fully.

Ruby Mega Menu is fully responsive. All menu types which can be seen in the horizontal menu demo is responsive for mobile devices.

Each parent menu can be either clicked to go to an URL, or expanded to make its sublevels visible. This feature also covers blog sub-categories and shopping sub-categories. Test it now!

Bootstrap?
Ruby Mega Menu is a non-Bootstrap menu. Good news: yet, ruby menu is also compatible with Bootstrap.
Auto-height
Mega menu content heights are calculated automatically. So, you don't need to define height values manually.
675 Icons
With FontAwesome integration, ruby menu offers ~675 icons to be used in your menu, anywhere you want.
How to use
You will get a detailed documentation about usage. Any other questions will be answered for supported users.
